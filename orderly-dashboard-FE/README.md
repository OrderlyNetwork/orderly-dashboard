# orderly-dashboard-FE
## Install deps
```shell
npm install
```
## Start
MacOS or Linux
```shell
DEBUG=myapp:* npm start
```
Window
```shell
set DEBUG=myapp:* & npm start
```
Visit on explorer  
http://localhost:3000/example.html
